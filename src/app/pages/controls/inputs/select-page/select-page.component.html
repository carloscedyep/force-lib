Select 
Campo con items seleccionables, agrupados en un array de cualquier tipo, asegurandose de que tengan dos atributos: value y text.
El control tiene las siguientes entradas.
id (string): identificador del control.
placeholder (string): texto que se mostrará en el select cuando está vacio.
isReadonly (boolean): indica si el select es de solo lectura. Por defecto es false.
value (string): valor a mostrar en el select.
textAlign (string): alineación del select (left: a la izquierda, center: al centro, right: a la derecha). Por defecto es left.
label (string): etiqueta del select.
lenght (string): indica el tamaño del select (full, xxlarge, xlarge, large, medium, norma, small y tiny). Por defecto es normal
color (string): color del borde (primary, light y warn). Por defecto es light.
isInvalid (boolean): indica si el control del select es inválido.
haveFilter (boolean): indica si se muestra el filtro. Por defecto es falso.
hint (string): cadena de select que indica si el input tiene un estado inválido.
fixed (boolean): indica si el select está dentro de un container fixed, para que el menu a mostrar sea ubicado correctamente.
items (array): items que se mostrarán en el select.

Datepicker 
Simple campo de texto, no se recomienda el uso para campos numéricos, telefónicos o email, para esos campos revisar los controles específicos.
El control tiene las siguientes entradas.
color (string): color del borde (primary, light y warn). Por defecto es light.
hint (string): cadena de texto que indica si el input tiene un estado inválido.
id (string): identificador del control.
isFixed (boolean): indica si el control está dentro de un contenedor de posicionamiento fijo. Por defecto es false.
isInvalid (boolean): indica si el control del texto es inválido. Por defecto es false.
label (string): etiqueta del input.
lenght (string): indica el tamaño del input (full, xxlarge, xlarge, large, medium, norma, small y tiny). Por defecto es normal
placeholder (string): texto que se mostrará en el input cuando está vacio.
textAlign (string): alineación del input (left: a la izquierda, center: al centro, right: a la derecha). Por defecto es left.
value (string): valor a mostrar en el input.
Se recomienda el uso de un idControl único para asegurarse un buen funcionamiento de la página. Mas información en este link.

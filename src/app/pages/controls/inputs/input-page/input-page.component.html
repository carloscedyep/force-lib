Input 
Simple campo de texto, no se recomienda el uso para campos numéricos, telefónicos o email, para esos campos revisar los controles específicos.
El control tiene las siguientes entradas.
color (string): color del borde (primary, light y warn). Por defecto es light.
hint (string): cadena de texto que indica si el input tiene un estado inválido.
idControl (string): identificador del control.
isInvalid (boolean): indica si el control del texto es inválido.
isReadonly (boolean): indica si el input es de solo lectura. Por defecto es false.
label (string): etiqueta del input.
lenght (string): indica el tamaño del input (full, xxlarge, xlarge, large, medium, norma, small y tiny). Por defecto es normal
placeholder (string): texto que se mostrará en el input cuando está vacio.
prefix (string): texto que se muestra antes del input.
sufix (string): texto que se muestra después del input.
textAlign (string): alineación del input (left: a la izquierda, center: al centro, right: a la derecha). Por defecto es left.
type (string): tipo de input (text, password). Por defecto es text.
value (string): valor a mostrar en el input.
Se recomienda el uso de un idControl único para asegurarse un buen funcionamiento de la página. Mas información en este link.
Se recomienda el uso de un label para asegurarse un buen funcionamiento de la página. Mas información en este link.

</div>
TypeScript 
import { Component, OnInit } from '@angular/core';
 
import { FormBuilder, Validators } from '@angular/forms';
 
@Component({
 
   selector: 'app-input-page',
 
   templateUrl: './input-page.component.html',
 
   styleUrls: ['./input-page.component.scss']
 
})
 
export class InputPageComponent implements OnInit {
 
  inputFormGroup = this.formBuilder.group( {
 
      input: ['', Validators.required]
 
  })
 
  constructor(private formBuilder: FormBuilder) { }
 
  ngOnInit(): void { }
 
}

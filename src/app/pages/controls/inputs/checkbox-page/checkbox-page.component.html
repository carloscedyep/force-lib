Checkbox 
Control clickeable que establece dos estados: checkeado y no checkeado.
El control tiene las siguientes entradas.
color (string): color del checkbox (primary, light y warn). Por defecto es light.
idControl (string): identificador del control.
isReadonly (boolean): indica si el checkbox es de solo lectura. Por defecto es false.
prefixText (string): etiqueta izquierda del checkbox.
sufixText (string): etiqueta derecha del checkbox.
value (string): valor a mostrar en el checkbox.
Se recomienda el uso de un idControl único para asegurarse un buen funcionamiento de la página. Mas información en este link.

import { Component, OnInit } from '@angular/core';
 
import { FormBuilder, Validators } from '@angular/forms';
 
@Component({
 
   selector: 'app-checkbox-page',
 
   templateUrl: './checkbox-page.component.html',
 
   styleUrls: ['./checkbox-page.component.scss']
 
})
 
export class CheckboxPageComponent implements OnInit {
 
  checkboxFormGroup = this.formBuilder.group( {
 
      checkbox: [false, Validators.required]
 
  })
 
  constructor(private formBuilder: FormBuilder) { }
 
  ngOnInit(): void { }
 
}

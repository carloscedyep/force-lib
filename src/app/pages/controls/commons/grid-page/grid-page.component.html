Grid 
Maquetación básica, donde se usan los siguientes estilos.
row : clase que distribuye los elementos en una fila.
xstart : alinea los elementos al inicio del eje x.
xcenter : alinea los elementos al centro del eje x.
xend : alinea los elementos al final del eje x.
xspace : alinea los elementos a todo lo largo del row.
ystart : alinea los elementos al inicio del eje y.
ycenter : alinea los elementos al centro del eje y.
yend : alinea los elementos al final del eje y.
ybaseline : alinea los elementos en base al texto interno de cada elemento.
column : actualiza la clase row, cambiando la dirección de los elementos de fila a columna.
w-full : clase que alarga el ancho del elemento al 100%.

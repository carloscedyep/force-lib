Accordion

Es un contenedor, que maneja la siguiente estructura:
<force-accordion> (etiqueta): etiqueta contenedora del componente.
<force-accordion-item> (etiqueta): etiqueta contenedora de cada item del componente. Debe estar dentro de la etiqueta <force-accordion> .
<force-accordion-label> (etiqueta): etiqueta de cabecera para cada item, donde se pueden colocar todo tipo de componentes. Debe estar dentro de la etiqueta <force-accordion-item> .
<force-accordion-body> (etiqueta): etiqueta para el contenido de cada item, donde se pueden colocar todo tipo de componentes. Debe estar dentro de la etiqueta <force-accordion-item> .

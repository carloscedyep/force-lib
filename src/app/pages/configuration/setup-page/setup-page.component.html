Instalación
Para utilizar esta librería se tiene que copiar la carpeta dentro de la carpeta scr de tu proyecto.
También se debe copiar los archivos dentro de la carpeta assets de tu proyecto.
Luego se debe agregar las siguientes lineas en el archivo angular.json.
angular.json
También se debe actualizar el archivo styles.scss.
styles.scss
Finalmente se deben agregar las siguientes lineas en el archivo tsconfig.json.
tsconfig.json
Con todas estas configuraciones el proyecto está listo para usar la librería.

Instalación
Para utilizar esta librería se tiene que copiar la carpeta force-lib dentro de la carpeta scr de tu proyecto.
También se debe copiar los archivos iconos dentro de la carpeta assets de tu proyecto.
Luego se debe agregar las siguientes lineas en el archivo angular.json. 
...
 
"styles": [
 
   ...
 
   "src/assets/css/uicons-regular-rounded.css",
 
   "src/assets/css/uicons-solid-rounded.css"
 
],
 
...
También se debe actualizar el archivo styles.scss. 
@import 'app/force-lib/styles/styles.scss';
Finalmente se deben agregar las siguientes lineas en el archivo tsconfig.json. 
...
 
"paths": {
 
   ...
 
   "@force/controls/*":["app/force-lib/controls/*"],
 
   "@force/models/*":["app/force-lib/models/*"],
 
   "@force/directives/*":["app/force-lib/directives/*"],
 
   "@force/services/*":["app/force-lib/services/*"]
 
},
 
...
Con todas estas configuraciones el proyecto está listo para usar la librería.
